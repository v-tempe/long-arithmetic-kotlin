# Длинная арифметика на Kotlin
## 1. 🏷 Краткое описание
Данный проект представляет собой 
реализацию длинной арифметики натуральных чисел
на языке программирования Kotlin.


## 2. 🚀 Пример работы
Далее приведён простой пример
использования данного класса на практике:
````kotlin
val a = BytePerDigit("99999999999999999999")
val b = BytePerDigit("1")

println(a + b)  // 100000000000000000000
println(a.compareTo(b))  // >= 1

for (digit in a) {
    print(digit)  // 999999...
}
````

## 3. 🔑 Ключевые возможности
- ✅ Конструктор из String с валидацией и нормализацией
- ✅ toString(), digitCount, iterator()
- ✅ equals(), hashCode() (согласованные)
- ✅ Операторы +, -, compareTo
- ✅ Property-based тестирование через Jqwik 
(инварианты, оракул BigInteger)
- ✅ Полная неизменяемость, thread-safe


## 4. 📐 Архитектура и ограничения
- Только натуральные числа (≥ 0). Отрицательные не поддерживаются.
- a - b бросает IllegalArgumentException, если a < b.
- Внутреннее представление: база 10, 0 кодируется пустой коллекцией цифр.
- Объекты неизменяемы. Все операции возвращают новые экземпляры.
- Производительность: сложение/вычитание O(max(len(a), len(b))), хеш вычисляется при создании.


## 5. 🧪 Тестирование
Для тестирования используется библиотека Jqwik (property-based тесты).
